## react-three-egg（蛋仔派对three.js版）

### 项目介绍

<img src="cover.png" alt="cover.png" style="zoom: 67%;" />  

利用`react` + `threejs`技术栈构建的第三人称Q版闯关类游戏

### 操作

行走：`↑` / `↓` / `←` / `→`（方向键）  

跳跃：`空格`

### 技术栈
- `react`
- `three`
- `@react-three/fiber` 
- `@react-three/rapier`
